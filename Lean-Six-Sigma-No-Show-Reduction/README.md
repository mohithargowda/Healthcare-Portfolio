# Dental No-Show Reduction Project

## Objective
Reduce patient no-show rates using Lean Six Sigma methodologies.

## Tools Used
- DMAIC Framework
- Pareto Analysis
- Fishbone Diagram
- 5 Whys
- Control Plan

## Key Findings
- Forgetfulness was the leading cause of no-shows.
- Appointment conflicts and transportation issues were major contributors.

## Proposed Improvements
- SMS reminders
- Same-day confirmations
- Waitlist management
- Flexible rescheduling

## Skills Demonstrated
- Process Improvement
- Quality Improvement
- Data Analysis
- Root Cause Analysis
