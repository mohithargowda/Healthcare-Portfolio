# Healthcare Cost & Workflow Efficiency Dashboard

## Project Overview
Built a healthcare operations dashboard to analyze where outpatient clinics may lose time and revenue.

## Key Metrics Analyzed
- Completed visits
- No-show rates
- Patient wait times
- Staff overtime
- Estimated revenue
- Lost revenue from no-shows

## Key Insight
Missed appointments are not only an operational issue. They also create measurable financial impact through lost revenue and inefficient clinic workflow.

- Microsoft Excel

- KPI analysis
- Operational trend analysis
