# Operational Gaps in Post-Surgical Outpatient Recovery

## Project Overview
Created a healthcare operations case study focused on care continuity, escalation design, patient navigation, and workflow optimization.

## Problem Identified
The case showed how fragmented post-surgical follow-up can force patients to coordinate their own recovery, referrals, authorizations, and escalation needs.

## Proposed Solution
A Post-Surgical Recovery Continuity Pathway with:
- Risk flagging
- Coordinator assignment
- Closed-loop referral tracking
- Escalation protocol

## Dashboard Concept
Designed a Recovery Navigator Dashboard to track active recovery cases, escalation needs, referral delays, urgent care diversions, and coordinator work queues.

## Skills Demonstrated
- Care coordination analysis
- Workflow redesign
- Patient navigation strategy
- Operations case study development
- Healthcare project planning
